# AI-Customer-Support-Chatbot

## Overview
This project is an AI-powered customer support chatbot designed to handle
real-time user queries such as order status, refunds, delivery issues,
payments, and account support.

The chatbot is deployed on Telegram and provides instant responses using
intent-based natural language processing.

## Features
- Greeting and welcome message
- Order status and delivery queries
- Refund and return policy support
- Payment and account issue handling
- Smart fallback for unrecognized queries
- 24/7 automated customer support

## Tools & Technologies
- Dialogflow (Intent-based NLP)
- Telegram Bot API

## Dataset
Customer Support Conversations Dataset (Kaggle)  
Used for designing realistic customer queries and intents.

## How It Works
1. User sends a message on Telegram
2. Dialogflow processes the message
3. The appropriate intent is matched
4. A response is sent back to the user
5. Fallback is triggered if no intent matches

## Result
A fully functional AI chatbot capable of handling common customer support
queries efficiently.
